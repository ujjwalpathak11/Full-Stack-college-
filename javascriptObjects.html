1. An alert box is a pop-up window that displays a message to the user. 
    
<script>
alert("Your message goes here");
<script>

  2. A prompt box is a pop-up window that displays a message to the user and allows them to enter input 
    
<script>
prompt('Write your name')
<script>

  3. A confirm box is a pop-up window that displays a message to the user and asks for their confirmation. 
    
<script>
confirm('Press a button')
<script>

  4. `window.open(url, name, specs, replace)`: Opens a new browser window or tab with the specified URL, name, and window features. The `replace` parameter specifies whether to replace the current page in the browser history.`window.open(url, name, specs, replace)`: Opens a new browser window or tab with the specified URL, name, and window features. The `replace` parameter specifies whether to replace the current page in the browser history. 
    
<script>
window.open("", "", "width=200,height=100");
<script>

  5. window.close()`: Closes the current browser window or tab.
    
<script>
window.close();
<script>

  6. window.print() method opens the print dialog box and allows the user to print the current page. 
    
<script>
window.print();
<script>
